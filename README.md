# Radiology-Performance

This repository contains the data and R scripts used in the Master's thesis entitled:

**The impact of heavy medical equipment on the economic performance of private radiology centers: a panel data analysis of centers supported by Vidi Capital.**

## Repository content

- `HALAS.xlsx` : dataset used for the empirical analyses.
- `MSC THESIS.Rmd` : R script used for data preparation, econometric analyses, tables and figures.

The repository is provided to ensure the reproducibility of the empirical results presented in the thesis.
